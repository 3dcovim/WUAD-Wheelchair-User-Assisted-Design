VR software enables the evaluation of architectural designs from the perspective of wheelchair users through an immersive virtual environment. Its main features include:

Immersive VR navigation of buildings using a wheelchair simulator synchronized with the virtual environment.

Integration with BIM/CAD models, allowing architects to import 3D building designs for evaluation.
