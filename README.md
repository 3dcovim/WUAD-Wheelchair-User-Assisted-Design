
# WUAD-Wheelchair-User-Assisted-Design V0.9

VR software enables the evaluation of architectural designs from the perspective of wheelchair users through an immersive virtual environment.

# Features
The current version of this Unity project provides the following features:
* Immersive VR navigation of buildings using a wheelchair simulator synchronized with the virtual environment.
* Integration with BIM/CAD models, allowing architects to import 3D building designs for evaluation.
* Interactive exploration tools, including opening doors and interacting with scene elements.
* User feedback system based on markers, allowing wheelchair users to place annotated markers indicating accessibility issues (e.g., manoeuvrability, slope, steps, reachability, furniture distribution).
* In-VR measurement tools to measure distances and evaluate spatial constraints.
* Mini-map interface with teleportation, providing a top-down view of each floor and enabling quick navigation between locations.
* Bidirectional communication workflow between designers and end users, enabling iterative accessibility improvements.
* Real-time interaction with motion-tracking systems, including head tracking, wheelchair motion capture, and hand tracking devices.
